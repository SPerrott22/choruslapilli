Yay, now we have a single state of the Board which is an array of 9 values corresponding to the values of the squares.

We can create a function handClick that modifies a given element of the array using the setSquares function and then pass this handleClick function to each Square for it's specific index. Notice that we use the arrow-function notation because if we pass in handleClick(0) for example, the parens will cause it to be called right away as part of the Board's rendering process leading to an infinite loop.

onClick is a built-in component of the button element. But onSquareClick prop could be named anything really. It is conventional to use onSomething for props which represent events and handleSomething for the function definitions that handle those events.

The reason we used slice to create a copy of squares array is because we want to maintain immutability benefits that will allow us to use cool feautres like time travel in the future. We could naturally have just done squares[i] = "X"; since you can actually modify the contents of a const array in JS so long as you aren't reassigning the whole array. But this would cause future enhancements to be more difficult. For instance, by default, all child components re-render when a parent component's state changes. Immutability allows us to easily compare whether a child component's data has changed or not and skip re-rendering those children for better performance.

function calculateWinner(squares) {
  const lines = [
    [0, 1, 2],
    [3, 4, 5],
    [6, 7, 8],
    [0, 3, 6],
    [1, 4, 7],
    [2, 5, 8],
    [0, 4, 8],

    if (move > 0) {
      description = 'Go to move #' + move;
    } else {
      description = 'Go to game start';
    }
    return (
      <li key={move}>
        <button onClick={() => jumpTo(move)}>{description}</button>
      </li>
    );
  });

  return (
    <div className="game">
      <div className="game-board">
        <Board xIsNext={xIsNext} squares={currentSquares} onPlay={handlePlay} />
      </div>
      <div className="game-info">
        <ol>{moves}</ol>
      </div>
    </div>
  );
}

function calculateWinner(squares) {
  const lines = [
    [0, 1, 2],
    [3, 4, 5],
    [6, 7, 8],
    [0, 3, 6],
    [1, 4, 7],
    [2, 5, 8],
    [0, 4, 8],
    [2, 4, 6],
  ];
  for (let i = 0; i < lines.length; i++) {
    const [a, b, c] = lines[i];
    if (squares[a] && squares[a] === squares[b] && squares[a] === squares[c]) {
      return squares[a];
    }
  }
  return null;
}
